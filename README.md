## 🎯 项目简介

**VitalityLab Pro** 是一个基于现代Web技术构建的综合性健康科学平台，融合循证医学、分子生物学、生物化学和营养科学，为用户提供从宏观体征到微观代谢的全方位健康分析。

### 核心使命

- 🧬 **科学普及**：将复杂的生物医学知识转化为可视化的交互体验
- 📊 **精准评估**：基于WHO、NIH、哈佛医学院等权威机构标准
- 🔒 **隐私优先**：所有计算在本地完成，数据永不离开设备
- 🎮 **趣味学习**：通过游戏化设计让健康知识变得生动有趣

### 适用人群

| 用户类型 | 使用场景 |
|---------|---------|
| 👤 普通用户 | 日常健康监测、体重管理、饮食规划 |
| 🏃 健身爱好者 | 体脂追踪、训练计划、营养补充 |
| 🩺 健康从业者 | 客户评估工具、健康教育素材 |
| 🎓 学生/教师 | 生物医学教学、生理机制演示 |
| 🔬 研究人员 | 快速计算工具、数据可视化参考 |

## ✨ 功能特性

### 1. 智能健康评估系统

#### 基础指标计算
- **BMI (身体质量指数)**：基于WHO亚洲标准分类
- **BMR (基础代谢率)**：Mifflin-St Jeor方程，精度±5%
- **TDEE (每日总能量消耗)**：基于活动系数动态计算
- **BFP (体脂百分比)**：Deurenberg公式 + 美国海军 circumference 法
- **IBW (理想体重)**：BMI 18.5-24范围计算
- **METs (代谢当量)**：相对代谢强度评估

#### 进阶人体测量
- 腰围/腰臀比（中心性肥胖评估）
- 颈围（睡眠呼吸暂停风险）
- 体脂分布分析（Android vs Gynoid型）

### 2. 交互式人体解剖

| 器官系统 | 包含内容 | 科学深度 |
|---------|---------|---------|
| 🧠 神经系统 | 脑代谢、神经递质、血脑屏障 | 细胞生物学级别 |
| ❤️ 心血管 | 心肌代谢、电生理、冠脉循环 | 生理生化级别 |
| 🫁 呼吸系统 | 肺泡气体交换、血红蛋白氧合 | 生物物理级别 |
| 🫀 消化系统 | 酶动力学、pH调节、吸收机制 | 分子生物学级别 |
| 🦴 肌肉骨骼 | 肌丝滑动理论、能量系统 | 生物力学级别 |
| 🧪 肝脏代谢 | 尿素循环、解毒机制、胆汁合成 | 生物化学级别 |

| 类别 | 元素 | 关键功能 |

**5阶段完整消化流程**

```
口腔 (0-5min) → 食道 (5-10s) → 胃 (2-4h) → 小肠 (3-5h) → 大肠 (12-48h)
```

每阶段包含：
- ⏱️ 精确时间参数
- 🧪 pH值与酶活性
- ⚗️ 生化反应方程式
- 📏 解剖结构参数

### 5. 智能推荐引擎

基于用户档案生成个性化方案：

**目标导向算法**
- ⚖️ **保持模式**：维持TDEE，均衡宏量营养素

- 🏃 **运动表现**：周期化营养，训练窗口补充

**动态调整因素**
- 年龄相关代谢下降（每十年-2-3%）
- 性别差异（女性脂肪分布、男性肌肉量）
- 体成分目标（减脂保肌 vs 增肌减脂）

## 🔬 科学原理

### 计算公式详解

#### 1. BMI (Body Mass Index)
```
BMI = weight(kg) / height(m)²

WHO亚洲标准：
- < 18.5：偏瘦（营养不良风险）
- 18.5-23.9：正常
- 24.0-27.9：超重
- ≥ 28.0：肥胖

局限性：不区分脂肪与肌肉质量
```

#### 2. BMR (Basal Metabolic Rate)
**Mifflin-St Jeor方程**（1990，精度最高）

```
男性：BMR = 10W + 6.25H - 5A + 5
女性：BMR = 10W + 6.25H - 5A - 161

其中：
W = 体重(kg)
H = 身高(cm)
A = 年龄(岁)

误差范围：±5%（vs 间接测热法）
```

#### 3. TDEE (Total Daily Energy Expenditure)
```
TDEE = BMR × 活动系数

活动系数：
- 久坐（1.2）：办公室工作，无运动
- 轻度（1.375）：每周1-3天轻度运动
- 中度（1.55）：每周3-5天中度运动
- 高度（1.725）：每周6-7天高强度运动
- 极高（1.9）：体力劳动者/运动员
```

#### 4. BFP (Body Fat Percentage)

**Deurenberg公式**（基于BMI估算）
```
BFP = (1.20 × BMI) + (0.23 × Age) - (10.8 × Sex) - 5.4

Sex：男性=1，女性=0

误差：±3-5%（vs DEXA扫描）
```

**美国海军Circumference法**（更精确）
```
男性：BFP = 495/(1.0324 - 0.19077×log₁₀(Waist-Neck) + 0.15456×log₁₀(Height)) - 450

女性：BFP = 495/(1.29579 - 0.35004×log₁₀(Waist+Hip-Neck) + 0.22100×log₁₀(Height)) - 450

误差：±1-3%（需准确测量）
```

### 生物化学机制

#### 能量代谢
```
葡萄糖氧化：
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 38ATP (理论)
实际产量：30-32 ATP（考虑质子泄漏）

脂肪氧化：
C₁₆H₃₂O₂ (棕榈酸) + 23O₂ → 16CO₂ + 16H₂O + 129ATP
```

#### 蛋白质合成信号
```
mTORC1激活条件：
1. 亮氨酸 > 3g（传感器：Sestrin2）
2. 胰岛素/IGF-1信号（PI3K-Akt通路）
3. 机械张力（细胞骨架变形）

合成速率：0.04-0.06 g/kg/h（最大）
```

### 代码规范

#### JavaScript
```javascript
/**
 * 计算基础代谢率
 * @param {number} weight - 体重(kg)
 * @param {number} height - 身高(cm)
 * @param {number} age - 年龄(岁)
 * @param {string} gender - 性别 ('male'|'female')
 * @returns {number} BMR值(kcal)
 * @throws {Error} 当参数无效时抛出
 */
function calculateBMR(weight, height, age, gender) {
    // 参数验证
    if (weight <= 0 || height <= 0 || age <= 0) {
        throw new Error('生理参数必须为正数');
    }
    
    // 计算逻辑
    const sexFactor = gender === 'male' ? 5 : -161;
    return (10 * weight) + (6.25 * height) - (5 * age) + sexFactor;
}
```
